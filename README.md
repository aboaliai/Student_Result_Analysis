## 📋 About Student_Result_Analysis

Student_Result_Analysis is a menu-driven system that helps you check student results from CSV files. It uses simple tools to give clear information about performance. You can see grades, find the top students, and check scores by subject. You do not need programming skills to use this software.

The system runs on Windows and shows results in an easy-to-understand way. It works with data stored in common CSV files. The program uses Python libraries like Streamlit and Pandas, but you do not have to install or learn those. Everything you need is included when you download it.

## 🎯 How to Use Student_Result_Analysis

### 1. Load Your Data

- Prepare your student data in a CSV file. Make sure the file has columns for student names, subjects, and scores. A common format includes columns like:  
  - Student Name  
  - Subject  
  - Score  

- In the program window, find the "Load Data" menu item. Click this to open a file selector.

- Browse your computer for the CSV file and select it.

### 2. View Grade Summary

- After loading data, select the "Grade Summary" option. This will automatically assign grades based on scores. 

- Grades follow a standard scale like A, B, C, D, or F.

### 3. Find Top Performers

- Click the "Topper Identification" menu item.

- The program shows the best students based on total scores or averages. It shows the top 3 or top 5, depending on your choice.

### 4. See Subject-wise Performance

- Pick the "Subject Analysis" option.

- The system calculates average scores and shows how the class did in each subject.

- You can view summary tables or charts to better understand the data.

### 5. Export Results

- Use the "Export Results" feature to save your analysis.

- Exported files come in CSV or Excel format.

- This lets you share reports with others or keep records.
